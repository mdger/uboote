Detailed Description

The GameController encapsulates the LevelController. It is responsible for keeping the actual state of a level and keeping track of the player score and delegates events to the GameRenderer if needed. e.g. When the level is completed, the game needs to transition to the next level and tells the GameRenderer to present the current level in the status bar
